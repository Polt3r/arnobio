# Módulo 01 — Fundamentos

Este módulo introduz os primeiros elementos formais
da programação.

Aqui começamos a traduzir pensamento em código.

## Objetivo do módulo

- Entender como dados entram e saem
- Trabalhar com variáveis
- Compreender tipos de dados
- Operar valores de forma lógica

## Linguagem utilizada

Python, apenas como meio.
O foco é o raciocínio.

## Regra

Se você não consegue explicar o código em palavras,
você não o entende.
